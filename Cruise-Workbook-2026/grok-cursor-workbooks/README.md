## 📋 Project Overview

A collection of fun, educational Python apps for cruise ship passengers and crew. 
The project demonstrates vibe coding workflows on **iPad** using Grok Work and **Mac** using Cursor, both use the same **DigitalOcean Ubuntu Droplet** as the backend.

## 🛠️ Toolchain (Both workflows share the same Ubuntu Droplet)

**iPad Workflow** (Grok Build):
- Blink Shell + Mosh (terminal)
- Working Copy (Git)
- Grok (this chat) for vibe coding

**Mac Workflow** (Cursor):
- Cursor AI Editor + Composer
- Warp Terminal
- GitHub sync

**Shared Backend**:
- DigitalOcean Ubuntu 24.04 Droplet
- Python venv + rich, pandas, numpy, matplotlib, etc.

## 🚀 How to Get Started

1. **Create the Ubuntu Droplet** on DigitalOcean (1-2 GB RAM recommended)
2. **Clone this repo** on the Droplet
3. **Setup venv** and install requirements
4. **Choose your workflow**:
   - iPad → Use Mosh in Blink + this chat with Grok
   - Mac → Open in Cursor + Warp terminal
5. Start building! Say "Build utils.py" to Grok or use Composer in Cursor.
