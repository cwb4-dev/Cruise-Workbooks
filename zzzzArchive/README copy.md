## 🚀 Featured Projects

### 🎲 Monte Carlo Simulation
A deep dive into Monte Carlo methods for simulation and probabilistic modeling:
- Python notebooks with simulation code
- Real-world examples in finance, physics, and decision-making
- Interactive visualizations of convergence and uncertainty
- [Read the Article: Monte Carlo Methods Decoded](./Monte_Carlo_Simulation/Monte%20Carlo%20Methods%20Decoded.md)

### 🧠 Game Theory
Exploring strategic decision-making with a focus on:
- Game theory fundamentals and Nash equilibria
- Intersection with Markov Chain Monte Carlo (MCS) methods
- Interactive notes and strategy analyses
- Economics and behavioral science applications

### 🛒 Costco Price Auditor
A practical automation tool:
- Track and log price changes over time
- Identify trends for smarter purchasing decisions
- Python scripts and Claude-based code playbook

### 🥧 pi-mono
Computational approaches to calculate π:
- Monte Carlo estimation (dart-throwing method)
- Series expansions and numerical algorithms
- Comparison of accuracy and convergence rates

### 🤖 AI Learning Workbooks
Growing collection of AI/ML experiments:
- Hands-on model implementations
- Prompt engineering examples
- Integration with various AI models and APIs

### 📊 FFT Workbook
Signal processing fundamentals:
- Frequency analysis and transformations
- Noise filtering techniques
- Practical applications in time-series data

### 🗓️ Cruise Workbook 2026
Travel planning and documentation:
- Detailed itinerary and port research
- Comprehensive packing lists
- Travel tips and logistics

## 🔧 Prerequisites & Setup

### Required Tools
```bash
# Clone the repository
git clone https://github.com/cwb4-dev/Cruise-Workbooks.git
cd Cruise-Workbooks

# Each project has its own dependencies - check individual READMEs
```

### Common Dependencies
- Python 3.8+
- Jupyter Notebook / JupyterLab
- NumPy, Pandas, Matplotlib, Seaborn
- Additional libraries per project

### Quick Start
```bash
# For Monte Carlo Simulation
cd Monte_Carlo_Simulation
pip install -r requirements.txt  # (if exists)
jupyter notebook

# For Game Theory
cd "Game Theory (deepseek-based)"
# Open Obsidian notes or run Python scripts
```
